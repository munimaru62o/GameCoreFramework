# AI Usage Policy

### ✓ Code Analysis & Reading (Permitted)
All users are fully permitted and encouraged to use AI assistants (e.g., ChatGPT, GitHub Copilot, Claude) to analyze, explain, or understand the code, architecture, and documentation within this repository. We welcome this as it enhances developer experience and overall efficiency.

### ✗ Model Training & Scraping (Strictly Prohibited)
Scraping, extracting, or using any source code, data, or documentation from this repository for the purpose of training, fine-tuning, or developing generative AI models, Large Language Models (LLMs), or any other machine learning algorithms is strictly prohibited. 

**This repository opts out of all automated datasets intended for AI training.**
